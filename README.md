# Генератор случайных паролей

Простая программа для генерации случайных паролей

## Описание

Данная программа генерирует случайные пароли, используя комбинацию букв, цифр и специальных символов

## Особенности

- ✅ Генерация паролей произвольной длины
- ✅ Использование всех категорий символов:
  - Буквы верхнего регистра (A-Z)
  - Буквы нижнего регистра (a-z)
  - Цифры (0-9)
  - Специальные символы (!@#$%^&*() и тд)
- ✅ Легкость в использовании и настройке

## Требования

- Python 3.6 или выше
- Используются только стандартные модули Python (`random`, `string`)

## Установка

1. Скачайте или склонируйте репозиторий
2. Убедитесь, что установлен Python 3.6+

## Использование

### Базовое использование

```python
from password_generator import generate_password

# Генерация пароля стандартной длины (12 символов)
password = generate_password()
print(f"Ваш новый пароль: {password}")

# Генерация пароля определенной длины
long_password = generate_password(16)
print(f"Длинный пароль: {long_password}")
```

### Прямой запуск

Отредактируйте файл `password_generator.py` чтобы изменить длину пароля:

```python
password_length = 16  # Измените на нужную длину
print("Ваш новый пароль:", generate_password(password_length))
```

Или запустите скрипт напрямую:

```bash
python password_generator.py
```

## Рекомендации по безопасности

- Используйте пароли длиной не менее 12 символов
- Для критически важных сервисов рекомендуется длина 16+ символов
- Храните пароли в надежном месте (менеджер паролей)
- Не используйте один и тот же пароль для разных сервисов
